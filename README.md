# 🐔 Le Poulailler

Application React + Vite pour compter les œufs pondus par vos poules, jour par jour.

## Installation

```bash
npm install
```

## Lancer en développement

```bash
npm run dev
```

Ouvrez http://localhost:5173

## Build production

```bash
npm run build
npm run preview
```

## Fonctionnalités

- Compteur journalier avec boutons + / −
- Appui long pour compter rapidement
- Historique avec graphique des 7 derniers jours
- Export CSV
- Export Excel (.xlsx)
- Données sauvegardées localement (localStorage)
